# WAPlus Setup Guide: "The One-Click Follow-Up"

**Goal:** Admin gets a lead, clicks ONE button, and sends the beautiful Course Invite + Payment QR code.

---

## Part 1: Installing the "Superpower"

1.  **Open Chrome** on your Desktop/Laptop (this works best on Computer).
2.  Go to the **Chrome Web Store**.
3.  Search for **"WAPlus - CRM for WhatsApp"** (or Cooby/WA Web Plus).
4.  Click **"Add to Chrome"**.
5.  Open **web.whatsapp.com** and scan your QR code to login.
6.  *You will now see new Tabs (All, Unread, Business) and sidebar tools.*

---

## Part 3: The 3-Second Workflow (How Admin Uses It)

**Scenario:** Dr. Sharma messages "I want to join."

1.  **Open Chat:** Admin opens Dr. Sharma's chat.
2.  **Type `/`**: Type a forward slash in the message box.
3.  **Select `Invite_GenAI`**: The list pops up. Click your saved template.
4.  **Send:**
    *   Dr. Sharma instantly gets:
        1.  The Professional "Digital Invite Card" Image.
        2.  The QR Code (in the image).
        3.  The Text Link.
5.  **Tag User:**
    *   Click the **Tag Icon** (top of chat).
    *   Select **"Sent Payment Link"**.
